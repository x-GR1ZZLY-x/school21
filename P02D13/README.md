# P02D13 — "Игра жизни" Конвея на C с ncurses

## Цель проекта
Реализовать консольную симуляцию "Игры жизни" Джона Конвея для поля 25×80 клеток.  
Программа отображает живые клетки (`*`) и мёртвые (пробел) в терминале.

## Описание
- Поле: 25 строк × 80 колонок  
- Входной файл содержит начальное состояние:  
  - `1` — живая клетка  
  - `0` — мёртвая клетка  
- Пользователь может управлять скоростью симуляции:  
  - `a` / `A` — увеличить скорость (уменьшить задержку)  
  - `z` / `Z` — уменьшить скорость (увеличить задержку)  
- Программа завершается при нажатии пробела  
- Поле циклическое (границы "закольцованы")

## Сборка и запуск
Перед компиляцией убедитесь, что установлена библиотека ncurses.  
Компиляция:

```bash
cd P02D13
gcc -o life src/main.c -lncurses
./life input1.txt
