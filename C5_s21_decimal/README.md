# C5_s21_decimal — Реализация decimal библиотеки

## Цель проекта
Реализовать собственную библиотеку для работы с типом decimal, обеспечивающим высокую точность вычислений с фиксированной точкой. Библиотека предназначена для финансовых и других критичных к точности расчетов задач. Проект соответствует стандарту C11, POSIX.1-2017 и оформлен в соответствии с Google Style Guide.

## Описание
- s21_decimal.h Основной заголовочный файл с описанием типа и функций
- Арифметика Реализация базовых операций (сложение, вычитание, умножение, деление)
- Сравнение Функции сравнения decimal-чисел
- Преобразования Конвертация между int, float и decimal
- Дополнительные функции Округление, усечение, смена знака
- Тестирование Unit-тесты с покрытием не менее 80% и отчетом через gcov

## Особенности реализации
```
typedef struct {
    int bits[4];
} s21_decimal;
```
Структура хранения  
bits[0–2] — 96-битная мантисса  
bits[3]  
	биты 16–23 — степень (scale, от 0 до 28)  
	бит 31 — знак (0 — плюс, 1 — минус)  

Диапазон значений
от -79,228,162,514,264,337,593,543,950,335
до +79,228,162,514,264,337,593,543,950,335

Используется банковское округление (round half to even)
