# C2_SimpleBashUtils — Реализация утилит cat и grep

## Цель проекта
Реализовать кроссплатформенные аналоги системных утилит cat и grep в стиле POSIX с поддержкой всех стандартных и GNU-специфичных флагов. Проект соответствует стандартам C11 и POSIX.1-2017, разработан в соответствии с Google Style Guide и поддерживает комбинирование флагов в точности как оригинальные утилиты.

## Описание
- s21_cat: Полная реализация утилиты cat с поддержкой всех флагов (включая GNU-специфичные -b, -e, -n, -s, -t)
- s21_grep: Реализация утилиты grep с поддержкой флагов -e, -i, -v, -c, -l, -n и полной поддержкой комбинаций флагов
- Общая библиотека: Модуль file_utils для работы с файлами, используемый обеими утилитами
- Тестирование: Автоматические тесты для проверки корректности работы с разными комбинациями флагов

## Сборка проекта

Для cat:

```bash
cd src/cat
make
./s21_cat [флаги] [файлы]
```

Для grep:

```bash
cd src/grep
make
./s21_grep [флаги] [шаблон] [файлы]
```
