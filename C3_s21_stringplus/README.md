# C3_s21_stringplus — Расширенная библиотека обработки строк

## Цель проекта
Реализовать расширенную версию стандартной библиотеки обработки строк string.h с полной поддержкой всех стандартных функций и дополнительными возможностями. Проект соответствует стандарту C11 и POSIX.1-2017, разработан в соответствии с Google Style Guide и включает тестовое покрытие.

## Описание
- s21_string.h: Полная реализация стандартных функций обработки строк с префиксом s21_
- s21_sprintf: Расширенная реализация функции форматирования строк с поддержкой спецификаторов и флагов
- Специальные функции: Дополнительные функции обработки строк, вдохновленные классом String в C#
- Тестирование: Unit-тесты с покрытием более 90% кода и отчет о покрытии через gcov

## Сборка проекта

Сборка библиоткеи:
```bash
make
```
Запуск тестов:
```bash
make test
```
Генерация отчета о покрытии тестами:
```bash
make gcov_report
```
