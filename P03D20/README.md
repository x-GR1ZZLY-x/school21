# P03D20 — Программа для построения графиков математических функций

## Цель проекта
Реализовать программу на C для построения графика математической функции,
введённой пользователем в виде выражения.

Программа:
- парсит выражение
- переводит его в постфиксную форму (обратная польская запись)
- вычисляет значения функции
- отображает график в консоли

---

- `-`
- `*`
- `/`

### Функции
- `sin`
- `cos`
- `tan`
- `ctg`
- `sqrt`
- `ln`

### Переменные
- `x`

### Скобки
- `(`
- `)`

---

## Принцип работы

1. Пользователь вводит выражение (например: `sin(x)`).
2. Выражение разбирается на токены (числа, операторы, функции).
3. Инфиксная запись переводится в постфиксную (алгоритм сортировочной станции).
4. Для каждого значения `x` вычисляется результат.
5. Строится график в области:
   - Ширина: 80 символов
   - Высота: 25 символов
   - Диапазон x: от 0 до 4π
   - Диапазон y: от -1 до 1

Если выражение некорректно — программа выводит сообщение n/a и завершает работу.
